# Cloud Backup Utility

Консольная утилита для резервного копирования файлов и папок в облачные хранилища.

## Возможности

- ✅ Резервное копирование локальных файлов и папок
- ✅ Поддержка нескольких облачных провайдеров:
  - Яндекс.Диск
  - Google Drive
  - Dropbox
  - AWS S3
- ✅ Инкрементное копирование (только измененные файлы)
- ✅ Шифрование данных перед загрузкой (опционально)
- ✅ Логирование операций
- ✅ Восстановление из резервной копии
- ✅ Планировщик автоматического резервного копирования

## Установка

### Требования
- Python 3.8+
- Зависимости: `pip install -r requirements.txt`

### Быстрая установка
```bash
git clone https://github.com/yourusername/cloud-backup-util.git
cd cloud-backup-util
pip install -e .

cloud-backup-util/
├── src/
│   ├── providers/     # Модули облачных провайдеров
│   ├── crypto/        # Шифрование и безопасность
│   ├── core/          # Основная логика
│   └── cli.py         # Интерфейс командной строки
├── tests/             # Тесты
├── requirements.txt   # Зависимости
